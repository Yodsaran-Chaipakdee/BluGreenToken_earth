03 · Why it matters

ทำไม beyond-carbon value สำคัญกับนักลงทุน

นักลงทุนไม่ได้มองแค่ปริมาณเครดิต แต่ต้องการเห็นคุณภาพของโครงการ ความโปร่งใสของข้อมูล และความพร้อมในการติดตามผลระยะยาว

เห็นบริบทของพื้นที่จริง

ตัวเลขทางน้ำ พื้นที่ชุ่มน้ำ อาคาร ถนน และแนวชายฝั่งช่วยให้เห็นว่าโครงการตั้งอยู่ในภูมิทัศน์แบบใด และมีองค์ประกอบใดรอบพื้นที่ปลูก

เพิ่มความโปร่งใสของ project story

Beyond-carbon layer ช่วยอธิบายว่าพื้นที่โครงการมีความหมายมากกว่าคาร์บอนอย่างไร โดยแยกบทบาทของ carbon asset กับข้อมูลบริบทของพื้นที่อย่างชัดเจน

ต่อยอดเป็นแผนติดตามผล

ข้อมูลเชิงพื้นที่ทำหน้าที่เป็นฐานสำหรับเลือกพื้นที่สำคัญ ลงตรวจภาคสนาม เก็บภาพหลักฐาน และจัดทำ evidence room ให้ตรวจสอบได้ต่อเนื่อง

09 · Investor FAQ

คำถามหลักสำหรับนักลงทุน

Beyond-carbon value เกี่ยวข้องกับ token อย่างไร?

Token อ้างอิงสินทรัพย์หลักคือ carbon credit ส่วน beyond-carbon value เป็นข้อมูลเสริมที่ช่วยให้เห็นคุณภาพเชิงบริบทของพื้นที่โครงการ ความโปร่งใสของข้อมูล และแผนติดตามผลระยะยาว

ข้อมูลนี้ช่วยการตัดสินใจลงทุนอย่างไร?

ช่วยให้นักลงทุนเห็นว่าโครงการตั้งอยู่ในพื้นที่แบบใด มีองค์ประกอบด้านระบบน้ำ ชุมชน และชายฝั่งอย่างไร และ developer มีแนวทางสร้างหลักฐานเพิ่มเติมอย่างเป็นระบบหรือไม่

แผนที่ในหน้านี้มาจากไหน?

แผนที่ถูกดึงมาจากชุด deploy ของ project evidence เดิม และจัดใหม่ให้เป็น investor-facing layer โดยคัดเฉพาะภาพที่อ่านง่ายและมีความหมายต่อการเล่าเรื่องโครงการ

ข้อมูลเชิงเทคนิคอยู่ตรงไหน?

ข้อมูลเชิงเทคนิค แผนที่ methodology และ QA/QC ถูกจัดไว้ใน evidence room เพื่อให้ตรวจสอบได้โดยไม่ทำให้หน้า investor ซับซ้อนเกินไป

Disclosure

ตัวชี้วัด beyond-carbon ในหน้านี้มาจากการคัดกรองเชิงพื้นที่และการวิเคราะห์บริบทของโครงการ ใช้เพื่อเพิ่มความโปร่งใส วางแผนติดตามผล และช่วยให้นักลงทุนเข้าใจพื้นที่โครงการมากขึ้น ไม่ใช่คำรับรองผลตอบแทนทางการเงิน สภาพคล่องของ token หรือผลลัพธ์เชิงปริมาณด้านระบบนิเวศ ชุมชน หรือการป้องกันชายฝั่ง
